Javascript is an interpreted and single-threaded language.Javascript runs on clients browser . We know that every browser Have js engine(like: Google have V8 engine).Through JS engine we run our js codes and if everything works good it shows output.JS engine have Parser which checks JS codes line by line,if any error occurs it throw error and if not it create AST(Abstract Syntax Tree).Then it convert the code to Machine code.Then our system run the code and we have the output.

Javascript is a single-threaded language and also a combination of Synchronous and Asynchronous events.Event loops works as First Come First Serve.If the user clicks many buttons at a certain time,the thread will make a stack .And do task one after another,the stack is called Event Queue.It works until the Event queue is empty ,this is what the Event Loop do.Simple,it waits for the Event Queue to be clear again and repeats the process as same again and again.
